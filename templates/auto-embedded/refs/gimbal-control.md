# 云台稳定控制（二/三轴：角度环 + 角速度环串级，IMU 稳像）

> 本文件在以下情形按需加载：
> - 做相机/激光/枪管云台（RoboMaster 步兵云台、稳像吊舱、二轴指向）
> - 要"在空间中稳住指向"（抗底座扰动），而不只是"转到某个关节角"
>
> 这是串级控制 + 姿态估计 + 电机驱动的合体。姿态来源见 `mahony-ahrs-reference.md`；串级/PID 见 `pid-control-reference.md`/`balance-car-cascade-control.md`；电机见 `foc-control-overview.md`/`motor-drive-simple-actuators.md`；限位/安全见 `control-safety-limits.md`。

---

## 铁律

**云台稳定 = 串级：外环锁"空间绝对指向"（payload IMU 给绝对姿态）→ 内环用陀螺角速度做快速阻尼 → 电机力矩。目标是"稳住空间指向、抗底座晃动"，不是"转到某关节角"——所以外环反馈优先用装在负载上的 IMU 绝对角，而非电机编码器相对角。**

- 内环角速度（陀螺）必须比外环角度快很多（典型内环 1–5kHz、外环 100–200Hz），否则两环打架。
- 反馈源分工：**payload IMU**（陀螺+加计融合）给"指向得稳不稳"；**电机编码器**给"关节在哪、限位、多轴解耦"。两者都要。
- 增益/限幅/前馈系数都是标定项（`coding-standards.md` §4.1）。

## 两环结构

```
指向目标(世界系) ─►[外:角度环 PD/PI]─► 角速度参考 ─►[内:角速度环 PI]─► Iq*/电压 ─► 直驱BLDC(FOC) ─► 云台轴
        ▲ payload IMU 绝对角(Mahony)                ▲ 陀螺角速度                       │
        └────────────────── + 重力/cogging/底座运动 前馈 ──────────────────────────────┘
                            编码器: 关节角→多轴投影解耦 + 软/硬限位
```

- **外:角度环**：误差 = 目标指向 − IMU 实测指向 → 输出角速度参考。用 PD/PI（D 用陀螺角速度，别差分角度）。
- **内:角速度环**：误差 = 角速度参考 − 陀螺 → 输出力矩（FOC 的 `Iq*`，`Id*=0`）。PI 为主。

## 电机与驱动

- 云台多用**直驱力矩 BLDC（DD/gimbal 电机）**：低 KV、高极对数、零背隙、低速大扭矩——直接出力矩、无减速箱回差。
- 跑 **FOC 的"云台/电压模式"**（`Id=0`、低速可不做电流采样、用电压近似转矩）即可；要精确转矩/抗扰再上完整电流环（见 `foc-control-overview.md`）。
- 角度反馈用**磁编码器/绝对编码器**（关节角 + 上电绝对位置，省找零）。

---

## 前馈（让 PID 只补残差）

- **重力/不平衡补偿**：负载质心偏离转轴时有重力矩 `τ ≈ m·g·d·cos(θ)`（d=质心到轴距，θ=俯仰角=与水平夹角；θ=0 力臂水平→重力矩最大，θ=±90° 竖直→为 0），按当前角前馈补上——否则不同俯仰角要不同 Kp，调不平。（若把 θ 定义为与竖直方向夹角，则改用 sin(θ)。）
- **齿槽(cogging)补偿**：装配后扫一遍"转矩脉动 vs 转子位置"做表，运行期按位置减去（直驱低速时 cogging 很明显）。
- **底座运动前馈**：若有底座 IMU/已知底座角速度，预先反向补偿，比纯反馈抗扰更快（稳像核心）。

## 多轴解耦 / 投影（俯仰非零时必做）

二轴(yaw/pitch)在 pitch≠0 时，**yaw 轴电机转的不再等于世界系航向变化**（陀螺轴随俯仰倾斜）。需用编码器俯仰角 `θp` 把世界系指令/陀螺量**投影**到云台轴：

- 经典做法：按内旋 ZYX，用 `R(θp)` 把"期望角速度/指向"在世界系↔云台轴间变换（参考"二轴云台末端指向控制"）。
- 接近 **gimbal-lock**（俯仰 ±90° 附近）yaw 失去权限，要限位或单轴退化处理，别让指令爆掉。
- 不做解耦的症状：低头/抬头时 yaw 跟随变慢/变快、画面斜向漂。

## 调参顺序（自内向外）

1. **内角速度环**：外环开环/置零，给角速度阶跃，调到快速无超调（陀螺噪声大→PI 不加 D）。
2. **外角度环**：内环锁定后，加 Kp 跟踪指向、加阻尼；验证抗底座晃动（手晃底座看指向是否稳）。
3. 加**重力前馈**消除不同俯仰角的增益不一致；再加 cogging/底座前馈精修。
4. 一次一参（`control-loop-sign-debug.md`）。

---

## 限位与连续旋转

- **pitch** 有机械限位 → 配**软限位 + 接近减速**（见 `control-safety-limits.md`）。
- **yaw** 要连续 360° 旋转 → 用**滑环(slip-ring)** 传电/信号，或限制圈数 + 线缆管理；多圈要处理编码器跨圈累加。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 外环用编码器relative角当"稳像" | 编码器是关节相对角、抗不了底座晃；稳像外环要用 IMU 绝对角 |
| 不做多轴投影 | 俯仰非零时 yaw 走样、画面斜漂 |
| 无重力前馈 | 不同俯仰角增益不一致、调不平 |
| 内外环带宽不分离 | 打架振荡 |
| 差分角度做阻尼 | 噪声大；用陀螺原始角速度 |
| 忽略 gimbal-lock | ±90° 附近 yaw 失权、指令爆 |
| 直驱无前馈硬拉 Kp | 抖/啸叫；用前馈 + 适度增益 |
