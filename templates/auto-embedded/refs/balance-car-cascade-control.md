# 两轮平衡车串级控制（直立环 + 速度环 + 转向环，经典 PID 实战）

> 本文件在以下情形按需加载：
> - 做两轮自平衡车/平衡机器人，要用**手调串级 PID**（电赛最常用路线）
> - 出现"只 PD 直立站不住 / 低头加速冲出去 / 转向打架 / 越调越抖"
>
> 这是 `lqr-example-segway.md`（模型法 LQR）的**实战姊妹篇**：LQR 是"算系数"，本篇是"手调串级 PID"。
> 姿态来源见 `.auto-embedded/refs/mahony-ahrs-reference.md`；PID 基础见 `pid-control-reference.md`；符号见 `control-loop-sign-debug.md`。

---

## 铁律

**平衡车是串级：直立环最内最快、速度环居中、转向环并联。仅靠 PD 直立环站不住——必须有速度环。**

- **为什么仅 PD 直立不行**：角度 PD 能把倾角 θ **镇定到 0**（Kp 够大时无稳态倾角误差），但**车轮的速度/位置是不受约束的自由积分模态**——θ≈0 时电机几乎不用出力，任何初速度/扰动/建模误差都会被积成**持续匀速漂移**（车直立着却滑走、最终冲出）。角度环管不到这个模态；速度环通过**微调"期望平衡角" θ\***（让车稍前/后倾去主动加减速）把速度/位置漂移闭环掉。
- 三环**调参顺序固定**：先直立环（站住）→ 再速度环（不跑）→ 最后转向环。
- 符号是头号杀手（角度正负、陀螺方向、电机极性、速度环正反接），先按 `control-loop-sign-debug.md` 对照实验验证再调参。

## 三环结构

```
速度目标 ─►[速度环 PI]─► 期望平衡角 θ* ─►[直立环 PD]─┐
                                                      ├─► 直立输出 ──┐
            转向目标 ─►[转向环 PD(yaw)]───────────────────► 转向输出 ─┤
                                                                      ▼
                                          左轮 PWM = 直立 + 转向 ; 右轮 PWM = 直立 − 转向
```

> 三环输出在电机层叠加：**直立项左右同号、转向项左右反号**（速度项已并入直立环的 θ*）。

---

## 1) 直立环（最内、最快，跑在 5–10ms 中断）

角度 + 角速度的 **PD**（本质是对倒立摆的状态反馈）：

```
直立输出 = Kp·(θ − θ*) + Kd·gyro
  θ     = 融合倾角(Mahony/互补，来自 mahony-ahrs-reference.md)
  gyro  = 该轴角速度(直接用陀螺，天然是角度的微分，比差分干净)
  θ*    = 期望平衡角(由速度环给；机械中点附近)
```

- 物理意义：`Kp`=回复力（拉回竖直），`Kd`=阻尼力（抵消转动惯量、抑振）。重心越高 → `Kp` 越小、`Kd` 越大（惯量大）。
- **先调直立**：θ*=机械中点，速度/转向置 0；加 `Kp` 到能站但抖，加 `Kd` 压抖（`Kd` 过大高频啸叫则退）。能短暂站立即可进下一步。

---

## 2) 速度环（居中，输出修"期望平衡角"）

```
期望平衡角 θ* = θ_mech − ( Kp_v·e_v + Ki_v·∫e_v )      // e_v = 速度目标 − 实测速度
  实测速度 = 左右编码器均值(低通)
```

- 机理：要前进 → 速度环输出让 `θ*` **前倾一点** → 直立环为追这个前倾角而驱动车轮前进 → 实现"靠倾角换速度"。
- **极性最易错**：速度环输出对 `θ*` 的加/减方向必须让"车快了就回正、慢了就更倾"，接反会**正反馈冲出去**——务必对照实验验证。
- 用 **PI**（无 D，速度信号噪声大）；积分配抗饱和（见 `pid-control-reference.md`）；`θ*` 限幅（防过倾摔倒）。
- 速度环带宽**远低于**直立环（典型慢 5–10×），否则与直立环打架。

---

## 3) 转向环（并联，叠加左右轮差）

```
转向输出 = Kp_yaw·(yaw* − yaw) + Kd_yaw·yaw_rate     // 无转向指令时 yaw* 锁存进入瞬间航向
```

- 与全向底盘"锁头状态机"同思想（见 `omni-wheel-odometry.md`）：无转向指令时锁航向、有指令时跟随。
- 叠加为左右轮**反号**项；注意它与速度/直立的限幅协调（总 PWM 不溢出）。
- 无磁时 yaw 仍漂（回 `mahony-ahrs-reference.md` 铁律），短赛程可接受。

---

## 调参顺序（务必自内向外，单变量）

1. **直立环 PD**：速度/转向=0，θ*=中点；调 Kp 站住、Kd 压抖。
2. **速度环 PI**：直立锁定后，给速度=0 看车是否原地不漂；调 Kp_v（响应）、Ki_v（消漂，配抗饱和）；确认极性（推一下车，应自己回中而非冲走）。
3. **转向环 PD**：前两环稳后，加转向、验证左右反号与锁头。
4. 全程一次只改一个参数（`control-loop-sign-debug.md` 纪律）。

## 与 LQR 的关系

`lqr-example-segway.md` 用模型 + LQR 一次算出状态反馈增益矩阵 K（角度/角速度/位置/速度统一最优），避免手调耦合。本篇手调串级 PID 是**无精确模型时的实战路线**，两者解的是同一个倒立摆问题。要可复现/可整定增益、有 MATLAB → 走 LQR；快速上手/无模型 → 串级 PID。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 只 PD 直立不加速度环 | 角度被镇定但车速/位置是不受控的自由积分模态 → 匀速漂移滑走冲出 |
| 速度环极性反 | 正反馈、越跑越快冲出——对照实验验证 |
| 用差分角速度 | 噪声大；直接用陀螺原始角速度做 Kd |
| 带宽不分离 | 速度环太快与直立环打架振荡 |
| θ* 不限幅 | 速度环饱和 → 过倾摔倒 |
| 姿态滞后 | 互补/滤波延迟拖慢直立环——见 imu-gyroscope-checklist DLPF/采样 |
| 符号没先验 | 调参全白费（control-loop-sign-debug）|
