# 路径规划与避障（全局 A*/Dijkstra · 局部 DWA/APF · 与跟踪分层）

> 本文件在以下情形按需加载：
> - 要从"地图/障碍"算出一条路径或局部速度指令（不只是跟踪已知路径）
> - 选 A*/Dijkstra（全局）/ DWA / 人工势场（局部避障）
>
> 这是机器人栈的**规划层**，在感知(`line-follow-vision-sensing.md`/`ranging-localization-sensors.md`)与跟踪控制(`path-tracking-pure-pursuit-stanley.md`)之间。

---

## 铁律

**分层：全局规划（地图→一条最优路径）+ 局部规划/避障（沿全局路径走、实时躲动态障碍→输出速度指令）+ 跟踪控制（把路径/速度变舵量）。三层职责不同，别混。规划层产出"路径或 (v,ω) 指令"，交给 `path-tracking`/底盘执行。**

- 规划吃**位姿与地图/障碍**：位姿来自里程计/融合（`omni-wheel-odometry.md`/`imu-wheel-ekf-fusion.md`），障碍来自测距/视觉（`ranging-localization-sensors.md`）。
- MCU 算力有限：栅格别太大、A* 加堆优化、DWA 采样数适度；复杂全局 SLAM/规划常放上位机，MCU 跑局部避障即可。

---

## 全局规划（已知地图，离线/低频）

把环境建成**栅格地图**（free/occupied，可加膨胀层把障碍按车体半径外扩）。

- **Dijkstra**：无启发的最短路，遍历广、保证最优、慢。
- **A\***：Dijkstra + 启发 `f(n)=g(n)+h(n)`（g=已走代价，h=到目标估计，曼哈顿/欧氏，需 admissible 不高估）。用优先队列(堆)，比 Dijkstra 快很多，最常用。
- **Hybrid A\***：考虑车辆运动学（转弯半径约束）的连续状态 A*，给阿克曼/非完整车出可行（可跟踪）路径。
- 产出栅格路径后做**平滑**（去锯齿/降采样/样条），再交跟踪。

---

## 局部规划 / 避障（实时、高频）

### DWA（动态窗口法）

在**速度空间 (v,ω)** 采样，预测短时轨迹，选评分最高的：

```
动态窗口 = {速度上限 Vs} ∩ {本周期加速度可达 Vd} ∩ {能在撞障前刹停的可行 Va}
对窗口内每组 (v,ω)：前推一小段轨迹，按目标函数打分
  G(v,ω) = σ( α·heading + β·dist + γ·vel )
    heading = 朝目标的朝向得分    dist = 离最近障碍的间隙    vel = 速度(鼓励快)
选 G 最大的 (v,ω) 下发，每周期重算
```

- 天然考虑动力学约束、平滑、能躲动态障碍。`α/β/γ` 是标定权重（偏激进/保守）。
- 适合差速/全向；阿克曼要把窗口约束到可行转角。

### 人工势场（APF，最轻量）

目标产生**引力**、障碍产生**斥力**，合力方向即运动方向：

```
引力  U_att = ½·k·d_goal²            （F_att 指向目标）
斥力  U_rep = ½·η·(1/ρ − 1/ρ0)²  (ρ≤ρ0 时)   （F_rep 推离障碍，ρ0=作用半径）
合力 F = F_att + F_rep → 期望速度方向
```

- 极简、实时、好实现；**致命缺陷=局部极小**（引力斥力抵消卡住、障碍前震荡）→ 加随机扰动/沿墙/切向力逃逸，或只当 DWA 的辅助。

### 选型

| 方案 | 层 | 优 | 缺 | 适用 |
|---|---|---|---|---|
| Dijkstra/A* | 全局 | 最优、完备 | 静态地图、算力 | 已知地图找路 |
| Hybrid A* | 全局 | 含运动学、可跟踪 | 更重 | 阿克曼/泊车 |
| **DWA** | 局部 | 动力学+动态障碍 | 需地图/障碍、调权重 | 实时避障主力 |
| **APF** | 局部 | 极轻 | 局部极小 | MCU 轻量避障/辅助 |

## MCU 落地

- 栅格分辨率与内存权衡；A* 用二叉堆 + closed 集位图；只在重规划时跑（事件触发，非每周期）。
- 局部 DWA/APF 跑在中频（如 10–50Hz），消费实时障碍（测距/视觉），输出 (v,ω) 给跟踪/底盘。
- 复杂建图/全局放上位机，MCU 跑局部 + 跟踪（分工见 `comms-protocol-bus-reliability.md`）。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 规划/跟踪/控制混写 | 难调难复用；分层 |
| 障碍不按车体膨胀 | 路径贴边刮蹭；加膨胀层(车体半径) |
| A* 启发不 admissible | 失最优；h 别高估真实代价 |
| APF 局部极小没处理 | 卡死/震荡；加逃逸或换 DWA |
| DWA 不含刹停约束 | 高速冲进障碍；Va 必须保证能停 |
| 全局每周期重算 | 算力爆；全局事件触发、局部才高频 |
| 用错位姿/障碍坐标系 | 规划全偏；统一坐标系(见 control-loop-sign-debug) |
