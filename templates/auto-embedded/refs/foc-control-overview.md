## 铁律

**FOC 的本质 = 用转子电角度 θ_e 把"三相耦合的交流量"旋到与转子同步的 dq 直流量，于是能用最简单的 PI 像控直流电机一样：`iq` 控转矩、`id` 控励磁（通常令 `id=0`）。**

- 整个 FOC 成败先取决于 **θ_e 准不准**：电角度错 → Park 变换错 → 转矩全错（甚至反转/失控）。有感靠编码器/霍尔标定，无感靠观测器，见对应分篇。
- 三个环**串级**、带宽**逐级拉开**：电流环最内最快、速度环居中、位置环最外最慢。**带宽不分离 = 互相打架发散**。
- 所有 PI 增益、限幅、观测器增益、注入幅值**都是标定参数**：带单位+来源+测量方法+适用硬件，禁裸魔法数字（`coding-standards.md` §4.1）。

## 适用 vs 不适用

| 场景 | 用 FOC？ |
|---|---|
| PMSM / 高性能 BLDC，要平滑转矩、低噪声、宽调速、精确转矩/位置 | ✅ 首选 |
| 要电流闭环（限流/精确转矩）、要能零速带载（配无感 HFI / 编码器） | ✅ |
| 极低成本、只要转起来、对噪声/纹波不敏感（风扇、水泵） | ◻ 方波六步换向更省（无需电流环/坐标变换） |
| 有刷直流电机 | ❌ 直接 PWM 调压即可，无需 FOC |

> BLDC（梯形反电动势）也能跑 FOC，只是正弦假设有少量转矩脉动；PMSM（正弦反电动势）是 FOC 的理想对象。

## 完整算法流水线（一个电流环周期内，通常 = PWM 周期）

```
                         ┌──────────────── θ_e (转子电角度) ───────────────┐
                         │                                                 │
 三相电流采样            ▼                          电流 PI(dq)            ▼
 ia,ib(,ic) ─► Clarke ─► iα,iβ ─► Park ─► id,iq ─► [id*-id]→PID_d→vd ─► inv Park ─► vα,vβ ─► SVPWM ─► PWM ─► 三相桥 ─► 电机
                                          ▲          [iq*-iq]→PID_q→vq        ▲
                                          │            + dq 解耦前馈           │
              θ_e 来源：有感(编码器/旋变/霍尔) 或 无感(SMO/磁链观测/HFI)  ── 见 sensorless 篇
```

逐步（每步的公式/实现见各分篇，这里给骨架）：

1. **采样**：PWM 中心对齐时刻采两/三相电流（采样时机见电流环篇）。
2. **Clarke**（3 相→2 相静止 αβ，幅值不变约定，`ia+ib+ic=0`）：`iα = ia`，`iβ = (ia + 2·ib)/√3`。
3. **Park**（αβ→旋转 dq，用 θ_e）：`id = iα·cosθ_e + iβ·sinθ_e`，`iq = −iα·sinθ_e + iβ·cosθ_e`。
4. **电流 PI（dq 各一个）**：`vd = PID_d(id*−id) − ω_e·Lq·iq`，`vq = PID_q(iq*−iq) + ω_e·Ld·id + ω_e·ψf`（末项是解耦前馈，见电流环篇）。常令 `id*=0`。
5. **逆 Park**：`vα = vd·cosθ_e − vq·sinθ_e`，`vβ = vd·sinθ_e + vq·cosθ_e`。
6. **SVPWM**：把 (vα,vβ) 合成空间电压矢量 → 三相占空比 → 定时器 PWM。
7. 速度/位置环（更低频，外层）算出 `iq*`（和位置→速度），见速度/位置环篇。

## 三个串级环（从内到外，带宽逐级拉开）

| 环 | 控什么 | 频率/带宽 | 控制器 | 反馈来源 |
|---|---|---|---|---|
| **电流环（转矩环）** | `id,iq`（→转矩） | 最快，跑在 PWM/采样中断（如 10–40 kHz）；带宽 `ωcc ≈ 1/10~1/20 · 2π·f_pwm` | PI（不用 D，见电流环篇） | 相电流采样 |
| **速度环** | 转速 ω | 中，`≤ 1/5~1/10` 电流环带宽 | PI（+加速度前馈） | 编码器差分/观测器 |
| **位置环** | 角度/位置 | 最慢，`≤ 1/5` 速度环带宽 | P 或 PI（+速度前馈） | 编码器/多圈 |

> **控制模式**：只跑电流环 = **转矩控制**；+速度环 = **速度控制**；+位置环 = **位置控制**。三者外层包内层。调参也按此顺序：**先电流环、再速度环、最后位置环**（见速度/位置环篇）。

> **id 给定**：表贴式 PMSM(SPMSM, Ld≈Lq) 通常 `id*=0`（最大转矩电流比的特例）；内嵌式(IPMSM, Ld<Lq) 用 **MTPA**（给负 id 借磁阻转矩）、高速用**弱磁**（负 id 压反电动势）。入门一律 `id*=0`。

## 角度/转速来源（决定有感/无感）

| 方式 | 器件 | 零速 | 精度 | 成本 | 见 |
|---|---|---|---|---|---|
| 增量编码器 / 旋变 / 磁编码器 | AS5047/旋变/光电 | ✅ | 高 | 高 | calibration 篇（电角度零点对齐） |
| 霍尔 | 3×霍尔 120° | ✅(粗) | 低(60°离散，需插值) | 低 | calibration 篇（换向表+插值） |
| 无感-反电动势/磁链(SMO 等) | 无 | ❌(零速无反电动势) | 中高(中高速) | 最低 | sensorless 篇 |
| 无感-高频注入 HFI | 无(需凸极) | ✅ | 中(低速) | 最低 | sensorless 篇 |

## MCU 落地要点（Cortex-M4F / 电赛视角）

| 项 | 做法 |
|---|---|
| 电流环位置 | 跑在 **PWM 触发的 ADC 中断**里（与 PWM 同步采样），不在主循环 |
| 时间预算 | 整个 FOC ISR（采样→Clarke/Park→2×PI→SVPWM）在 M4F @168MHz 上控制在 **< PWM 周期的 30–50%**（如 20kHz→<25µs，参考 foc 设计：ISR < 40µs）|
| 三角函数 | `sinf/cosf` 或查表（一次算 θ_e 的 sin/cos 复用于 Park+逆Park）；定点平台用 CORDIC/查表 |
| 浮点 | M4F/M7 用 `float` + 硬件 FPU；M0/M3 无 FPU 用 Q15/Q31 定点（见 `coding-standards.md`）|
| 死区 | 互补 PWM 必须配死区 + 死区补偿（见 calibration 篇），否则过零畸变 |
| 采样同步 | ADC 触发对齐 PWM 中心（下桥全开窗口），避开开关噪声 |
