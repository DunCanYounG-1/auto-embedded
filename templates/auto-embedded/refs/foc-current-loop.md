# FOC 电流环（转矩环）：dq 模型 · PI 整定 · 解耦前馈 · SVPWM · 电流采样

> 本文件在以下情形按需加载：
> - 要整定 FOC 最内环（电流/转矩环）的 PI、做 dq 交叉耦合解耦
> - 电流环阶跃超调/振荡/跟踪慢、转矩脉动大、过零畸变、采样取不准
>
> 前置：流水线总览见 `.auto-embedded/refs/foc-control-overview.md`；θ_e 来源见 `foc-calibration-checklist.md` / `foc-sensorless.md`。

---

## 铁律

**电流环是 FOC 性能的地基：它的带宽决定速度/位置环能拉多高。先把电流环的 PI 用"零极点对消"算出来、再调外环——别拍脑袋。**

- 电流环用 **PI 不用 PID**：被控对象是一阶 `1/(R+Ls)`，D 项只会放大采样高频噪声、无收益（zhihu 共识）。
- dq 两轴**有交叉耦合**（`ω_e` 越高越强）：高速必须**解耦前馈**，否则 d/q 互相拽、动态变差。
- 输出是电压，**受母线电压限幅**：限幅 + 抗积分饱和是必须项，否则饱和后 PI 失控。

---

## 被控对象：dq 电压方程

PMSM 在 dq 旋转系（电角速度 ω_e）：

```
vd = R·id + Ld·(did/dt) − ω_e·Lq·iq            ← d 轴
vq = R·iq + Lq·(diq/dt) + ω_e·Ld·id + ω_e·ψf   ← q 轴
                              └── 交叉耦合 ──┘  └ 反电动势 ┘
```

把交叉项当扰动剥掉后，每轴就是一阶对象 `i/v = 1/(R + L·s)`（d 轴用 Ld、q 轴用 Lq）。

---

## PI 整定：零极点对消（首选，有解析解）

对象 `G(s) = 1/(R + L·s) = (1/L)/(s + R/L)`，极点在 `−R/L`。PI `C(s) = Kp + Ki/s = Kp·(s + Ki/Kp)/s`。
**令 PI 零点 = 对象极点**：`Ki/Kp = R/L`，对消后开环 = `Kp/(L·s)`，闭环就是一阶、带宽 `ωcc = Kp/L`。于是：

```
选定电流环带宽 ωcc [rad/s]   →   Kp = L·ωcc ,   Ki = R·ωcc
                                  （d 轴用 Ld，q 轴用 Lq）
```

- **带宽怎么选**：`ωcc ≈ (1/10 ~ 1/20)·2π·f_pwm`（采样=PWM 时）。例 f_pwm=20kHz → `ωcc ≈ 1000~2000·2π?` 取 ~1000–4000 rad/s 是常见区间（学术示例有取 3679 rad/s、相位裕度 40°，速度环再低一个数量级）。拉太高会被 PWM/采样延迟和噪声反噬。
- **温度补偿（可选但鲁棒）**：R 随温度变，保持带宽不变可让 `Ki` 跟踪 R：`Ki/Kp = R(T)/L`。VESC 的做法是**只补 Ki 不补 Kp**（Kp∝L 基本不随温变，Ki∝R 随温变）。
- 离散化：用后向欧拉/Tustin，`Ki_z = Ki·Ts`；注意积分项单独限幅。

> 也可用 MATLAB **FOC Autotuner** / 厂商工具（Microchip MCAF、ST MCSDK）自动整定，但要懂上面的解析关系才能判断结果是否合理。

---

## 交叉耦合解耦 + 前馈

PI 只负责把每轴对象当一阶 `R+Ls` 控；把耦合项与反电动势**前馈补掉**，让 PI 看到干净的一阶对象：

```
vd_cmd = PID_d(id* − id) − ω_e·Lq·iq
vq_cmd = PID_q(iq* − iq) + ω_e·Ld·id + ω_e·ψf
```

- 验证：把 vd_cmd 代回 dq 方程，耦合项正好抵消 → `(R+Ld·s)·id = PID_d 输出`，干净一阶。q 轴同理。
- `ψf`(永磁磁链)、`Ld/Lq` 来自参数辨识（见 calibration 篇）；不准则前馈不全，但 PI 仍能兜底（只是动态差）。低速可省解耦，高速务必加。

---

## 电压限幅 + 抗积分饱和

- **dq 圆形限幅**：可用电压幅值 ≈ `Vdc/√3`（SVPWM 线性区）。`√(vd²+vq²) ≤ Vmax`；超了按比例缩，或**优先保 vd（维持磁场/弱磁）、再分配 vq**。
- **抗 windup**：输出饱和时停止/回退积分（clamping 或 back-calculation）；SimpleFOC 里 `PID_current_q.limit = voltage_limit`。
- 上电/模式切换/目标突变时复位积分。

> SimpleFOC 参考起点：`PID_current_q.P≈5, I≈300~1000, D=0`，电流低通 `LPF_current.Tf≈0.005`，`PID.limit=voltage_limit`（具体随电机/采样率标定，别照抄）。

## SVPWM（空间矢量调制）

把 (vα,vβ) 合成为旋转电压矢量，用相邻两个基本矢量+零矢量在一个 PWM 周期内合成：

- **判扇区**（6 个 60° 扇区）→ 算两相邻矢量作用时间 T1/T2 → 加零矢量 T0 → 生成三相占空比。
- **中心对齐 PWM**（上下计数）+ 七段式：谐波小、开关均匀。
- 线性调制区电压利用率比 SPWM 高 ~15%（等效注入三次谐波/马鞍波）。过调制区要专门处理。
- 实现等价于"Clarke 反变换 + 零序注入"：`v_offset = −(max+min)/2` 注入三相后归一到 [0,1] 占空比——比查扇区更简洁，工程常用。

## 电流采样（最容易取不准的地方）

| 拓扑 | 说明 | 采样时机 |
|---|---|---|
| **Inline（相串联）** | 电阻串在相线，全程可测，精度高、成本高 | 较宽松 |
| **低边单电阻** | 母线回流单电阻，靠不同矢量分时重构三相 | 窗口窄，靠近 PWM 边沿，低调制比时不可测→需移相/最小窗口 |
| **低边双/三电阻** | 每相下桥串电阻，下桥导通时可测 | **PWM 中心（下桥全开）** 采，最稳 |

- **采样触发对齐 PWM 中心**（下桥导通窗口最宽、远离开关噪声）；ADC 由定时器硬件触发，别软件轮询。
- 三相只需采两相（`ic = −ia − ib`），但三相都采可做**和≈0 自检**（查零偏/坏通道）。
- 零偏标定 + 增益（shunt×放大倍数）见 `foc-calibration-checklist.md`。

## 验证（落地前必做）

- **电流环阶跃**：给 iq* 小阶跃，示波器看上升时间/超调（参考目标：上升 < 0.2ms、超调 < 10%）。
- **电流 THD**：稳态 FFT，目标 < 5%。
- **解耦有效性**：高速给 id* 阶跃，看 iq 是否被带动（耦合残留）。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 用 PID（带 D） | 一阶对象，D 放大采样噪声、无收益 |
| 不解耦 | 高速 dq 互拽、动态恶化、限幅时更糟 |
| 采样时机错 | 没对齐 PWM 中心 → 采到开关尖峰 → 电流噪声大、转矩脉动 |
| Kp/Ki 用错电感 | d 轴用 Ld、q 轴用 Lq；IPMSM 上 Ld≠Lq 别混 |
| 无抗饱和 | 电压饱和后积分爆掉、退饱和迟滞 |
| θ_e 不准 | 电流环再好也白搭——回 calibration/sensorless 篇 |
