# FOC 无感（无位置传感器）：SMO / 磁链观测 / HFI / I-F 启动与切换

> 本文件在以下情形按需加载：
> - 要省掉编码器，用电流电压估转子电角度 θ_e 与转速 ω_e
> - 纠结 SMO / 磁链观测器 / 高频注入 HFI，或卡在"零速启动 / 低速带载 / 观测器切换"
>
> 前置：流水线见 `.auto-embedded/refs/foc-control-overview.md`；电流环见 `foc-current-loop.md`；参数辨识见 `foc-calibration-checklist.md`。

---

## 铁律

**无感不创造可观测性。中高速靠反电动势（∝转速）能观出角度；零/极低速反电动势≈0、观不出来，必须靠"凸极效应(HFI)"或"开环硬拖(I-F)"。没有一种方法通吃全速域——工程是分段 + 平滑切换。**

- 观测器精度强依赖电机参数 `R, Ld, Lq, ψf`：参数误差（尤其电感）放大角度误差。先做参数辨识。
- 角度从观测量里提取**不要直接 atan2**（噪声大、跳变）：用 **PLL/锁相**平滑出 θ_e 与 ω_e。
- 切换（开环↔观测器、HFI↔反电动势）是无感最容易翻车处：必须做**平滑过渡**（角度对齐 + 加权融合）。

## 速度分段与方法选型

| 速度段 | 可用信息 | 主力方法 |
|---|---|---|
| 零速 / 极低速 | 反电动势≈0；只有**凸极**(Ld≠Lq) | **HFI 高频注入**（需凸极，IPMSM 友好）；无凸极则只能 I-F 开环 |
| 启动 | 无 | **I-F 开环爬速**（或预定位）→ 升到中速切观测器 |
| 中高速 | 反电动势显著 | **SMO / 磁链观测器 / 磁链观测**（反电动势法） |
| 全速域 | — | **HFI + SMO(超螺旋) 融合 + 平滑切换** |

## 中高速：反电动势 / 磁链观测

### 滑模观测器 SMO（最常用）

αβ 系电流观测器，用滑模项"逼出"反电动势：

```
观测：  L·(dî/dt) = v − R·î − z          (î = 估计电流, αβ 各一路)
滑模项：z = k·sat(î − i)                  (sat = 饱和/sign，边界层抗抖振)
收敛后 z ≈ 反电动势 e_αβ；  e_α=−ψf·ω_e·sinθ_e, e_β=ψf·ω_e·cosθ_e
取角：  θ_e 由 e_αβ 经 PLL 提取（不要直接 atan2）
```

- **优点**：对参数(R,L)不敏感、鲁棒、实现简单——工程主力高速观测器。
- **抖振**：`sign` 切换引入高频抖振 → 用**边界层(sat 代 sign)** 或 **超螺旋 STA(super-twisting)** 抑制；传统 SMO 还需 LPF + 相位补偿（LPF 滞后随频率变，要按 ω 补角）。
- 进阶：变增益超螺旋 STSMO + 自适应反电动势模型，全速域更稳。

### 磁链观测器

积分反电动势得定子磁链、再与永磁磁链比对求角；纯积分有**直流漂移/初值**问题 → 用 **SOGI / 带通 / 限幅积分** 替代纯积分。低速反电动势小，仍受限。

## 零/低速：高频注入 HFI

利用**凸极效应**(Ld≠Lq)：注入高频探测信号，高频电流响应里含转子位置信息。

- **注入形式**：旋转电压注入 / 脉振(d 轴)正弦或**方波注入**（方波频率高、带宽好，VESC/主流用方波）。
- **解调**：对高频电流响应做解调（外差/带通）→ 误差信号 → PLL 出 θ_e。
- **极性辨识(N/S)**：HFI 初始有 180° 二义性，需磁饱和判极性（注入 d 轴脉冲看饱和不对称）定 N 极。
- **零低速带载**：可叠加正向 `id` 注入提升带载能力与可观性（代价是铜耗）。
- **限制**：需要凸极（SPMSM 凸极弱→HFI 差）；有高频噪声/损耗；带宽被注入频率限制。

## 启动与切换（无感最易翻车）

1. **预定位**（可选）：给固定电压矢量把转子拉到已知角（10–50ms），代价是可能反转/冲击。
2. **I-F 开环爬速**：以渐增的电角频率 + 恒定 `iq`(电流闭环、角度开环)硬拖电机加速到能观测的速度（如额定 5–10%、几百 rpm）。
3. **切观测器**：达到阈值速度后，把开环角度**平滑对齐**到观测器角度（角度差加权过渡/斜坡），再切到观测器闭环。
4. **HFI↔反电动势切换**（全速域）：低速 HFI、中高速反电动势，在重叠区按速度加权融合两者角度，避免切换抖动。

> 切换瞬间是失步高发点：必须做角度对齐 + 渐变权重，**禁止硬切**。

## 参数敏感性

- 电感误差对角度误差的放大 > 电阻误差；高速参数漂移要自适应观测器或在线辨识。
- `ψf` 影响反电动势幅值标定与速度估计。
- 先按 `foc-calibration-checklist.md` 把 `R, Ld, Lq, ψf` 辨识准，再上观测器。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 期望零速无感纯靠反电动势 | 零速反电动势=0，观不出来；必须 HFI 或 I-F |
| 直接 atan2 取角 | 噪声/跳变 → 用 PLL |
| SMO 抖振未处理 | 用边界层/超螺旋 + LPF 相位补偿 |
| 磁链纯积分 | 直流漂移/初值发散 → SOGI/带通/限幅积分 |
| 硬切观测器 | 切换瞬间失步 → 角度对齐 + 加权过渡 |
| 无凸极硬上 HFI | SPMSM 凸极弱，HFI 信噪比差 |
| 参数不准 | 角度误差放大、低速尤甚——先辨识 |
