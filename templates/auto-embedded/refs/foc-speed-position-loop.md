# FOC 速度环 + 位置环（串级、带宽分离、前馈、抗饱和）

> 本文件在以下情形按需加载：
> - 电流环已整定好，要往外加速度环 / 位置环（速度控制、定位、伺服）
> - 出现"低速画龙 / 速度纹波 / 定位过冲 / 高增益振荡 / 级联积分饱和"
>
> 前置：电流环见 `.auto-embedded/refs/foc-current-loop.md`；总览见 `foc-control-overview.md`。

---

## 铁律

**串级控制的命门是带宽分离：内环必须比外环快 5–10 倍，外环才能把内环近似当"理想跟随"。带宽不分离 = 两环互相打架振荡。调参顺序永远是：电流环 → 速度环 → 位置环。**

```
位置环(P/PI) ──θ*──► 速度环(PI) ──iq*──► 电流环(PI) ──vq──► 电机
   最慢            ≤1/5 速度          ≤1/5~1/10 电流          最快
```

---

## 带宽分离经验

| 环 | 带宽 | 相对 |
|---|---|---|
| 电流环 | `ωcc`（如 1000–4000 rad/s） | 基准 |
| 速度环 | `ωcs ≤ (1/5 ~ 1/10)·ωcc`（如 ~300 rad/s） | 比电流低约一个数量级 |
| 位置环 | `ωcp ≤ (1/5)·ωcs` | 再低 |

> 数量级分离是经验下限：分离越大越稳但响应越慢；够用即可，别盲目拉高外环增益当"响应快"。

## 速度环

**对象**：电流环（近似 1 阶/纯延迟）+ 机械 `1/(J·s + B)`（转矩→转速）。控制器 **PI**：

```
iq* = PI_speed(ω* − ω) + iq_ff          // iq_ff = (J/Kt)·(dω*/dt) 加速度前馈(可选)
```

- **转速估计**：编码器位置差分 + 低通（最简，量化噪声大→需滤波）；或 **M/T 法**（高速测周期、低速测脉冲）；或观测器/锁相。差分噪声是低速画龙主因。
- **抗饱和**：`iq*` 受电流限幅（=转矩限幅）；饱和时停积分。这是过冲/退饱和迟滞的关键。
- **前馈**：加速度前馈（上式 iq_ff）显著降跟踪滞后；负载已知可加负载转矩前馈。
- **增益尺度**：`Kp_speed ≈ J·ωcs/Kt`、`Ki_speed = Kp_speed·ωcs/(带宽分离比)`——但 J/Kt 多不准，常实测整定（先 Kp 到临界振荡退半，再加 Ki 消静差）。

---

## 位置环

**对象**：速度环 + 一个积分（速度→位置）。因已含积分，外层常用 **P**（或小 I 消静差）：

```
ω* = Kp_pos·(θ* − θ) + ω_ff           // ω_ff = dθ*/dt 速度前馈
```

- **轨迹规划**：阶跃目标会冲——用**梯形/S 曲线**速度规划（远快近慢），末端减速段独立处理（呼应 `omni-wheel-odometry.md` 反例里"匀速指向目标"的批评）。
- **速度前馈** `ω_ff = dθ*/dt`：让位置环只补误差、不靠纯比例硬拽，大幅降滞后/过冲。
- 多圈位置要处理编码器跨圈累加（与里程计回绕同理）。

---

## 调参顺序（务必自内向外）

1. **电流环**：按 `foc-current-loop.md` 零极点对消算 Kp/Ki，阶跃验证（超调<10%）。
2. **速度环**：电流环锁定后，给速度阶跃；Kp 加到临界振荡退一半 → 加 Ki 消静差 → 加加速度前馈降滞后。
3. **位置环**：速度环锁定后，加 Kp_pos + 速度前馈 + 轨迹规划；看定位无过冲、无残差。

> 一次只调一环、一次只改一个参数（呼应 `control-loop-sign-debug.md`：单变量实验）。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 带宽不分离 | 外环追内环速度 → 振荡发散；先确认电流环够快 |
| 速度差分噪声 | 编码器量化 → 差分放大噪声 → 低速画龙；加滤波/M-T 法/观测器 |
| 级联积分饱和 | 内外环都带 I，饱和时若都不停积分 → 严重退饱和迟滞 |
| 位置环纯 P + 无前馈 | 跟踪滞后大、动目标拖尾；加速度/速度前馈 |
| 阶跃目标无规划 | 冲过头/震荡；用梯形/S 曲线 + 末端减速 |
| 跨圈不处理 | 多圈位置溢出跳变 |
