# 运动规划 / 轨迹生成（梯形 · S 曲线 jerk-limited · 点到点 · 多轴同步）

> 本文件在以下情形按需加载：
> - 要让电机/底盘/关节"平滑地"从 A 走到 B（定位、走位、关节运动）
> - 出现"阶跃指令冲过头/震荡、步进丢步、机械抖动"
>
> 规划是**前馈**：它给位置/速度环喂可行的轨迹，不替代反馈环。配合 `pid-control-reference.md`、`foc-speed-position-loop.md`、`motor-drive-simple-actuators.md`。

---

## 铁律

**别给控制环喂阶跃目标——会冲过头/震荡、让步进丢步、激起机械振动。用运动规划生成"位置/速度/加速度都连续可行"的轮廓，把规划出的 v、a 作为前馈喂进环里（远比硬拉 Kp 稳）。**

---

## 梯形速度规划（3 段，最简）

加速→匀速→减速，加速度恒定（分段常值）：

```
t_a = v_max / a                         // 加速时间
s_a = v_max² / (2a)                     // 加/减速各走的距离
若 s ≥ 2·s_a：能到 v_max → 有匀速段，t_c = (s − 2·s_a) / v_max
否则（短行程）：到不了 v_max → 退化成三角形，峰值 v_peak = √(a·s)
每拍：按当前段更新 v[k]、积分出 p[k]，喂给位置/速度环
```

- 优点：实现极简、时间最优（给定 a、v_max）。
- 缺点：加速段切换处**加加速度(jerk)无穷大** → 冲击、振动、激励结构共振。

---

## S 曲线（jerk-limited，7 段）

把加速度也做成连续斜坡（恒定 jerk `J`），消除梯形的 jerk 突变。七段：**加加速 → 匀加速 → 减加速 → 匀速 → 加减速 → 匀减速 → 减减速**。每段用恒 jerk 运动学积分：

```
a(t) = a₀ + J·t
v(t) = v₀ + a₀·t + ½·J·t²
p(t) = p₀ + v₀·t + ½·a₀·t² + (J/6)·t³
```

- 优点：加速度连续、**jerk 有界** → 平滑、减振、保护机械与负载、步进不易丢步、伺服跟踪更准。
- 代价：实现复杂（要解各段时长，短行程时某些段可能消失）、比梯形略慢。
- `J` 越大越接近梯形（快但抖）、越小越柔（慢但稳）——是标定项。

---

## 梯形 vs S 曲线

| 维度 | 梯形（T 形） | S 曲线（7 段） |
|---|---|---|
| jerk | 转折处∞ | 有界（恒 J）|
| 振动/冲击 | 大 | 小 |
| 时间 | 最优 | 略长 |
| 实现 | 简单 | 复杂（分段求时） |
| 适用 | 低速/容差/算力紧 | 高速/高精/怕振/护机械 |

> 起步用梯形够用；要平滑/减振/护机械/步进防丢步 → S 曲线。

---

## 点到点 + 多轴同步

- **点到点(PTP)**：给 (起点, 终点, v_max, a, [J]) 生成单轴轮廓。
- **多轴同步**：以**最慢轴的总时间**为基准，缩放各轴的 v_max/a 使所有轴**同时启停**（否则合成轨迹走样）；或共享同一归一化时间基 `s(t)∈[0,1]` 再映射到各轴行程。

## 在线 vs 离线

- **离线**：预生成整条轨迹（表/多项式），运行期查表。简单、确定。
- **在线**：每拍实时算下一点（目标可中途变）。适合目标动态变化/跟随；S 曲线在线实现要处理"运动中改目标"的相位衔接。

## 与控制环配合（关键）

规划产出的 **v\*、a\*** 不只是位置目标——把它们作**速度前馈、加速度前馈**喂进环：
```
速度环目标 = 规划 v* ;   iq*/PWM 前馈 += Ka·规划 a*（加速度前馈）
位置环目标 = 规划 p* ;   位置环 + 速度前馈 v*
```
这样反馈环只补残差，跟踪滞后和过冲都显著下降（见 `pid-control-reference.md` 前馈、`foc-speed-position-loop.md`）。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 喂阶跃目标 | 冲过头/振动/步进丢步——这正是要规划的原因 |
| 只做位置规划不给前馈 | 浪费规划；把 v*/a* 当前馈才发挥价值 |
| 短行程不退化处理 | 到不了 v_max 时梯形要退化三角形、S 曲线要砍段，否则超调/算错 |
| 多轴不同步 | 各轴独立 PTP → 合成轨迹走样；按最慢轴缩放 |
| jerk 设太大 | S 曲线退化成梯形、失去减振意义 |
| 与采样周期不对齐 | 规划步进 dt 必须 = 控制周期（来自定时器，不硬编码）|
