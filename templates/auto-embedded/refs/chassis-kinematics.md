## 铁律

**先在一处把坐标系/正方向/轮编号/单位钉死（同 `omni-wheel-odometry.md` 第 0 步），再写正解(逆运动学,控制下发)与反解(正运动学,里程计)，且二者必须互为逆运算。任何符号不一致 = 能走但里程计反向漂/转向相反。**

- 选型先于公式：差速最简、阿克曼适合高速车型、4WIS 机动最强但 8 自由度过驱动需防轮拖滑。
- 各轮速度/转角**限幅**要协调（合成不溢出），逆解出的轮速超限要按比例缩放（保方向）。

## 差速（两轮/四轮差速）

车体前向 `vx`、横摆 `ωz`，轮距 `W`，轮半径 `r`：

```
逆解(车体→轮，控制下发):
  v_left  = vx − ωz·W/2
  v_right = vx + ωz·W/2          轮角速度 ω_wheel = v_wheel / r
正解(轮→车体，里程计):
  vx = (v_left + v_right)/2
  ωz = (v_right − v_left)/W
转弯半径 R = vx/ωz（直行时 R→∞）；ωz>0 的转向方向由 W、轮编号约定定，按 control-loop-sign-debug 验证
```

- 四轮差速 = 同侧两轮同速（左两轮、右两轮各一组）；越野机动强但转向必打滑（无横向自由度）。

## 阿克曼（前轮转向 + 后驱，自行车模型）

前轮转角 `δ`、轴距 `L`、转弯半径 `R`（绕后轴）：

```
tan(δ) = L / R   →   R = L / tan(δ)
ωz = vx / R = vx·tan(δ) / L      逆解: δ = atan( L·ωz / vx )（vx→0 时 δ 不可解，低速特殊处理）
里程计(后轴中心): vx=轮速, ωz=vx·tan(δ)/L, 再旋到世界系积分(见 omni-wheel-odometry)
```

- **真阿克曼几何**：内外前轮转角不同以共圆心、避免拖滑：`cot(δ_outer) − cot(δ_inner) = W/L`（W=前轮距）。单舵机近似时用平均角 δ（小角误差可接受）。
- 有**最小转弯半径**（δ 有机械上限）→ 规划要尊重（Hybrid A*，见 `path-planning-obstacle-avoidance.md`）；不能原地自转。

## 舵轮 / 四轮独立转向（4WIS / swerve）

每个轮可独立**驱动 + 转向**。轮 i 在车体系坐标 `(xi, yi)`，车体指令 `(vx, vy, ωz)`：

```
逆解(每轮的速度矢量 = 平移 + 旋转分量):
  v_ix = vx − ωz·yi
  v_iy = vy + ωz·xi
  轮速  = √(v_ix² + v_iy²)
  轮转角 = atan2(v_iy, v_ix)
正解(里程计): 用各轮实测速度矢量最小二乘反推 (vx, vy, ωz)
```

- **8 自由度 / 3 期望 = 过驱动**：控制不当会轮拖滑/打架；逆解给的就是"无拖滑"解。
- **就近优化**：目标转角与当前差 >90° 时，**轮速取负 + 转角翻 180°**，少转、响应快（WPILib `optimize`）。
- 模式：纯平移(各轮同角)、原地自转(各轮切向)、边走边转(平移+旋转叠加)。转向电机有机械限位则限幅。

## 选型对比

| 底盘 | 自由度 | 转弯 | 优 | 缺 | 适用 |
|---|---|---|---|---|---|
| 两轮差速 | 2 | 原地可转 | 极简、便宜 | 无横向、易滑 | 室内/循迹小车 |
| 四轮差速 | 2 | 原地(打滑) | 机动、越野 | 转向必滑、耗能 | 复杂地形 |
| 阿克曼 | 2(受限) | 有最小半径 | 高速稳 | 不能自转、机构复杂 | 高速车型/泊车 |
| 全向/麦轮 | 3 | 任意 | 全向、低速灵活 | 滚子损耗、负载低 | 短距全向（见 omni 篇）|
| 舵轮 4WIS | 3 | 任意 | 机动最强 | 8DOF 过驱、贵、复杂 | 高机动平台 |

## 标定（落地前必做）

- 几何量：轮距 `W`、轴距 `L`、轮径 `r`、各轮坐标 `(xi,yi)`——实测，别照抄图纸（装配误差）。
- `COUNTS_PER_MM`（编码器→线速度）直线测距标定；舵轮的**转向零位**逐轮标（同 FOC 电角度零点思路）。
- 正反解**数值往返**自检 + 直线/闭合测试（见 `omni-wheel-odometry.md` 验证法、`control-loop-sign-debug.md` 极性矩阵）。
- 阿克曼/4WIS 的 **IMU↔车体安装角**会污染里程计/NHC，见 `imu-wheel-ekf-fusion.md`。

## 常见陷阱

| 陷阱 | 说明 |
|---|---|
| 正反解符号不一致 | 能走但里程计反漂/转向反；往返自检 |
| 阿克曼用单一转角忽略内外差 | 低速拖滑（小角可接受，大角要 cot 差）|
| 阿克曼低速 δ=atan(L·ωz/vx) 除零 | vx→0 特殊处理（停车/原地不可解）|
| 4WIS 不做就近优化 | 转向电机来回大转、响应慢 |
| 轮速逆解超限不缩放 | 削顶失真、方向变；按比例整体缩放保方向 |
| 几何量照抄图纸 | 装配误差→里程计系统性偏；实测标定 |
